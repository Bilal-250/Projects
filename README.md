Premier League Data Exploration & Analysis (1993–2024)

Bilal Mohammed

------

Project Overview
This project analyses English Premier League standings data from 1993 to 2024 to identify the key factors that influence team success and evaluate how competitiveness in the league has evolved over time.

------

Objectives
- Identify the characteristics of high-performing teams
- Analyse long-term trends in league competitiveness
- Compare performance between top clubs and the rest of the league
- Build an interactive dashboard to communicate insights effectively

------

Tools
- CSV / Excel – Data handling and export
- SQL – Data querying, cleaning, and transformation
- Tableau – Dashboard creation and data visualisation

------

Dataset
Source: Kaggle – English Premier League Standings
Time period: 1993–2024

------

Analysis Performed

SQL
- Identified title winners and how many titles each club has won
- Found the best and worst single seasons by points and goals
- Analysed top 4 finishes and relegation trends across 30 seasons
- Compared clubs head to head using self-joins (e.g. champion vs runner-up each season)
- Filtered and aggregated data to answer specific questions about club and season performance

Tableau
- Built interactive dashboards to explore team performance
- Created scatter plots to analyse attack vs defence
- Visualised trends in points over time for top teams
- Designed filters for dynamic user interaction

Key Insights
- Teams finishing in the top 4 consistently show strong attack and defensive balance
- A small group of clubs dominate high point totals and top finishes over time
- The gap between top teams and mid-table teams has shown periods of increase, indicating fluctuating competitiveness

------

Dashboard Features
- Interactive filters for team and season selection
- Scatter plot showing goals scored vs goals conceded
- Line chart tracking team performance over time
- Summary visualisations of league dominance and trends
